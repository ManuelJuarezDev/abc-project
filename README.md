# Proyecto ABC - Gestión de Empleados

Sistema de gestión de empleados desarrollado con **Java 21**, **Spring Boot** y **PostgreSQL**.

## Cómo ejecutar
1. Es necesario tener Docker instalado.
2. Es necesario estar posicionado en la carpeta del proyecto (cd .\main)
3. Ejecutar comando:
   docker compose up --build
